# SRFP Eight Week Report

Welcome to the SRFP Eight Week Report project. This repository contains the LaTeX source for the internship report and a generated PDF version for easy viewing.

## 📄 What’s in this repository

- [report.tex](report.tex) — the main LaTeX source file for the report
- [final_report.pdf](final_report.pdf) — the compiled PDF output
- [README.md](README.md) — project guide and usage instructions

## ✨ Project Overview

This project is designed to help you:

- organize the final internship report clearly
- edit the content in LaTeX with ease
- generate a polished PDF version for submission or sharing

## 🛠️ How to edit and compile the report

1. Open [report.tex](report.tex) in your editor.
2. Make the necessary changes to the report content.
3. Compile the LaTeX file to generate the PDF.

### Compile command

If you have LaTeX installed, run:

```bash
pdflatex report.tex
```

If needed, run it twice to ensure references and page numbers are updated properly.

## 💡 Recommended tools

For a smoother editing experience, consider using:

- VS Code
- LaTeX Workshop extension
- A LaTeX compiler such as TeX Live or MiKTeX

## ✅ Quick workflow

- Edit the report in [report.tex](report.tex)
- Compile it to update [final_report.pdf](final_report.pdf)
- Review the result and share the PDF

## 📌 How to commit your changes

Run these commands in the project folder:

```bash
git status
git add README.md report.tex final_report.pdf
git commit -m "Update report README and content"
git push origin main
```

If your branch is named differently, replace `main` with your branch name.

## 🙌 Notes

Keep the report well-structured, proofread, and updated regularly so the final PDF stays clean and professional.
